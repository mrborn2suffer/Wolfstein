Wolfstein Raycasting Engine using Java 
- 2.5D rendering technique for java
- Helps in understanding how java visualizes 3D when in reality it's all 2d
- Based on POV, distance from user is close then object (wall) appears closer i.e big, if the user is far from object then object appears to be far i.e small.
- Disatnce is calculated based on some mathematical equations and values.
